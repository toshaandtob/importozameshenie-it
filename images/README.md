# Изображения для сайта

## Инструкция по загрузке изображений

### Рекомендуемые изображения для сайта:

1. **hero-image.jpg** - Главное изображение для главной страницы (рекомендуемый размер: 1920x1080px)
   - Тематика: технологии, импортозамещение, IT-инфраструктура

2. **import-substitution-concept.jpg** - Концепция импортозамещения (рекомендуемый размер: 1200x800px)
   - Можно использовать инфографику или схему

3. **russian-software.jpg** - Российское программное обеспечение (рекомендуемый размер: 1200x800px)
   - Примеры: логотипы российских IT-компаний, скриншоты ПО

4. **diagram.jpg** - Диаграмма или схема (рекомендуемый размер: 1000x800px)
   - Статистика, графики, схемы процессов

5. **placeholder.jpg** - Общая заглушка (рекомендуемый размер: 800x600px)

### Как загрузить изображения:

1. Нажмите кнопку "Upload files" или "Add file" -> "Upload files"
2. Перетащите ваши изображения в область загрузки
3. Добавьте описание коммита (например: "Add website images")
4. Нажмите "Commit changes"

### Важные требования:

- Формат файлов: JPG, PNG, WEBP
- Максимальный размер файла: 5 MB
- Используйте понятные имена файлов (латинскими буквами)
- Оптимизируйте изображения перед загрузкой

### Примечание:

После загрузки изображений, вы сможете использовать их на сайте по пути:
```html
<img src="images/hero-image.jpg" alt="Описание изображения">
```
